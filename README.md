# PYMESec - Frontend (Vite + React + Tailwind)

Proyecto generado para reproducir el estilo ASCII/monospace del dashboard solicitado.

## Instrucciones

1. Instalar dependencias:
```
npm install
```
2. Ejecutar en modo desarrollo:
```
npm run dev
```

Este proyecto usa datos estáticos y fuente monoespaciada (font-mono) para simular la estética de consola/ASCII.
